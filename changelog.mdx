<Update label="2026-01-09" description="v0.5.1" tags={["feature", "improvement"]}>

## New Features

- **GPU Analytics Dashboard**: Comprehensive market analytics with price tracking, availability trends, and GPU model comparisons across providers.
- **Enhanced AI Recommendations**: Improved AI assistant now considers your usage history and budget constraints for better GPU suggestions.
- **Async Order Creation**: New `/v1/marketplace/order/async` endpoint for reliable order creation with job status polling.

## Improvements

- Faster marketplace loading with optimized database queries
- Improved SSH connection reliability with automatic port allocation
- Enhanced provider verification system with reliability scoring
- Better error messages for payment and rental issues

## Bug Fixes

- Fixed occasional timeout issues during peak marketplace usage
- Resolved balance calculation discrepancies for long-running rentals
- Fixed session management issues causing unexpected logouts

</Update>

<Update label="2025-11-15" description="v0.4.0" tags={["feature", "crypto"]}>

## New Features

- **USDT Payments (TRC20)**: Deposit and withdraw funds using Tether on Tron network
- **Flexible Pricing Types**: Choose between per-minute, hourly, daily, weekly, or monthly billing
- **Provider Dashboard**: GPU owners can now manage their listings, view active rentals, and track earnings

## Improvements

- Real-time balance updates during rentals
- Improved marketplace filters with VRAM and location options
- Enhanced mobile responsiveness

## Bug Fixes

- Fixed provider port range conflicts
- Resolved rental duration calculation issues

</Update>
