## Common Topics

### Getting Started

- **How do I create an API key?** Go to **LLM API Keys** in the dashboard and create a new key.
- **What's the Python SDK?** Install with `pip install GPUniq` — see the [SDK docs](/sdk).
- **How do I add funds?** Go to Balance — deposit via USDT (TRC20), Stripe, or YooKassa.

### GPU Rentals

- **What are the three deployment modes?** [Marketplace](/gpu-marketplace) (pick a server), [Dex-Cloud](/gpu-dex-cloud) (pick GPU type), [Burst](/gpu-burst) (scale to 100 GPUs).
- **How do I connect?** Use SSH credentials from your dashboard. See [Managing Instances](/managing-instances).
- **How do I stop billing?** Click "Stop" to pause or "Delete" to terminate.

### Storage & Data

- **How do I persist data?** Use [Volumes](/volumes) — persistent storage that survives instance restarts.
- **How do I transfer files?** Upload via SDK/API or use SCP/rsync directly.
